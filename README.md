English Poem Writer (AI)

This is a simple web application that generates English poems using an AI API.
It is developed using HTML, CSS, and JavaScript.

Description

The user enters a topic or keywords.
The application sends the input to an AI API.
The AI returns a poem, which is shown on the screen.

Technologies Used

HTML
CSS
JavaScript
AI API (for poem generation)

How It Works

User types a theme or keywords
Clicks submit button
JavaScript sends the request to the AI API
The generated poem is displayed on the page

How to Run the Project

Download or clone the project
Open index.html in a web browser
Add your AI API key in the JavaScript file
Start generating poems
